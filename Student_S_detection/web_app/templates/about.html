About Student Stress Detection System

Overview

This system uses machine learning to predict student stress levels based on various factors including study habits, sleep patterns, social activities, and academic performance.

Features Analyzed

  • Study hours per day
  • Sleep hours per night
  • Exercise hours per week
  • Social activities frequency
  • Assignment deadlines
  • Exam pressure level
  • Family support level
  • Financial stress level
  • Academic performance
  • Workload level

Machine Learning Models

The system uses an ensemble of four machine learning models:

  • Logistic Regression: Linear classification model
  • SVM (Support Vector Machine): Non-linear classification
  • Naive Bayes: Probabilistic classifier
  • Random Forest: Ensemble of decision trees

The final prediction combines all models using ensemble learning for improved accuracy.

Stress Levels

  • Low: Well-managed stress, healthy balance
  • Medium: Moderate stress, room for improvement
  • High: High stress levels, requires attention
